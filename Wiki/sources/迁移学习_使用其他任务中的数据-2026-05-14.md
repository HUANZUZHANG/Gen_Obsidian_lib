# 迁移学习：使用其他任务中的数据

> 吴恩达机器学习课程 — 83.13.4 Transfer Learning
> 源视频：[B站链接](https://www.bilibili.com/video/BV16jyuBBEom/?p=83)

## 核心思想

当目标任务的数据量不足时，**迁移学习** 允许利用来自其他相关任务（大数据集）上训练好的模型，帮助目标任务取得更好的效果。

## 两步流程

### 1. 监督预训练 (Supervised Pre-training)

- 在一个**大型数据集**（如 100 万张猫、狗、车、人等图片，1000 个类别）上训练神经网络
- 训练完成后，网络的前几层已经学会了通用的图像特征
- 也可以直接**下载**研究人员已预训练好的模型（如 ImageNet、GPT-3、BERT）

### 2. 微调 (Fine-tuning)

- 复制预训练网络的参数（除输出层外）
- 去掉原来的输出层，替换为适应目标任务的新输出层
- 在新数据集上进一步训练

## 两种微调策略

| 策略      | 做法                    | 适用场景           |
| ------- | --------------------- | -------------- |
| **选项一** | 只训练新输出层的参数，冻结前几层      | 数据集非常小（几十到几百张） |
| **选项二** | 训练网络中所有参数，前几层用预训练值初始化 | 数据集稍大（几千张以上）   |
|         |                       |                |

## 为什么迁移学习有效？

神经网络分层学习的特征具有**通用性**：

- **第一层**：检测边缘（edge）
- **第二层**：组合边缘检测角点（corner）
- **第三层**：检测更复杂的形状（曲线、基本图形）
- 这些底层特征对**不同的计算机视觉任务都通用**

## 关键限制

- **输入类型必须一致**：预训练和微调的输入 $x$ 类型必须相同
  - 图像任务 → 需在图像数据上预训练的模型
  - 音频任务 → 需在音频数据上预训练的模型
  - 文本任务 → 需在文本数据上预训练的模型

## 实际效果

- 在百万级图像上预训练后，有时仅用 **50 张图**微调就能得到不错的结果
- 不是万能方法，但在数据集不够大时能显著提升效果

## 社区价值

- 研究人员共享预训练模型参数，任何人都可以免费下载使用
- 机器学习社区建立在彼此工作的基础上，开放共享让每个人都能做得更好
